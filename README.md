# Data Cleaning with Python

**Project Overview**

Data cleaning and standardization of a used car listings dataset using Python (pandas).

**Objective**

Prepare messy automotive listing data for reliable reporting and analysis.

**Tools Used**
* Python
* pandas
* Jupyter Notebook

**Key Cleaning Steps**
* Removed redundant index column
* Handled missing values <i>(Missing locations standardized to "Unknown")</i>
* Converted Year from float to integer type
* Split combined <i>Car Name</i> into structured <i>Brand</i> and <i>Model</i>
* Validated numerical ranges

**Files Included**
* [Cleaning.ipynb](cleaning.ipynb) - Full data cleaning process and transformation steps
* [cars24_cleaned.csv](cars24_cleaned.csv) - Final cleaned dataset ready for analysis
  
**Dataset**

Cars24 used car listings dataset taken from [Kaggle](https://www.kaggle.com/datasets/ujjwalwadhwa/cars24com-used-cars-dataset).
